CarGo

Project Name
CarGo

Description
CarGo is a web application built with Next.js and Bootstrap that allows administrators to manage a car 
rental service. Admins can add, edit, and delete car listings, while users can view available cars,
book them, and see their bookings. The application also includes a Home page and a Contact page for 
user interaction.

Features
- Admin Functionality: 
  - Add new cars to the inventory.
  - Edit existing car details.
  - Delete cars from the inventory.
  - View and update the bookings
- User  Functionality: 
  - View available cars with details.
  - Book cars for rental.
  - View booking history.
- Additional Pages: 
  - Home page for general information.
  - Contact page for user inquiries.

Technologies Used
- Next.js: A React framework for server-side rendering and static site generation.
- Bootstrap: A CSS framework for responsive design and styling.

How to Run the Project

Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
